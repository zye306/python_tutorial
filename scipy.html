SciPy Tutorial for Beginners#

Introduction#

SciPy (Scientific Python) is a Python library used for scientific and technical computing. It provides many efficient and user-friendly interfaces for tasks such as numerical integration, interpolation, optimization, linear algebra, and more.

Importing SciPy#

To use SciPy in your code, you need to import it. It’s also common to import NumPy since SciPy is built on it.

import numpy as np
import scipy

Basic Functions#

Integration#

SciPy provides functions to perform numerical integration. Here’s an example of definite integration using scipy.integrate.quad():

from scipy.integrate import quad

def f(x):
    return x**2

result, error = quad(f, 0, 1)
print("The integral result is", result)
print("The error is", error)

The integral result is 0.3333333333333333
The error is 3.700743415417188e-15

Optimization#

SciPy provides functions to perform optimization. Here’s an example of finding a local minimum using scipy.optimize.minimize():

from scipy.optimize import minimize

def f(x):
    return x**2 + 10*np.sin(x)

result = minimize(f, x0=0)
print("The minimum is at", result.x)

The minimum is at [-1.30644012]

Interpolation#

SciPy provides functions to perform interpolation. Here’s an example of 1D interpolation using scipy.interpolate.interp1d():

from scipy.interpolate import interp1d
from matplotlib import pyplot as plt

x = np.linspace(0, 10, num=11, endpoint=True)
y = np.cos(-x**2/9.0)
f = interp1d(x, y)

xnew = np.linspace(0, 10, num=41, endpoint=True)
ynew = f(xnew)   # use interpolation function returned by `interp1d`
plt.plot(x, y, 'o', xnew, ynew, '-')
plt.show()

This is a very basic introduction to SciPy. SciPy has many more features which you can explore in the official documentation.
